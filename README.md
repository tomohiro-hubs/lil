# Parent Company — 再生可能エネルギー戦略企業 コーポレートサイト

## プロジェクト概要

再生可能エネルギー事業の企画・設計・運用を統合的に支援する戦略企業のコーポレートサイトです。  
ディープフォレストグリーン × クールミントの自然系カラーパレットをベースに、アイソメトリックイラストと大型ウォーターマークセクションラベルを特徴とする、信頼感のある日本企業向けデザインです。

## 完成済み機能

### TOPページ (`index.html`)
- **HERO** — アイソメトリックイラスト付き左右分割レイアウト、装飾キューブ、スクロール誘導
- **SERVICE** — ウォーターマークラベル付き、01/02/03番号のスタッガード（非対称交互）レイアウト
- **PHILOSOPHY** — グリーン背景、縦書きラベル、明朝体見出し、都市イラスト
- **COMPANY (ABOUT US)** — 風力タービンイラスト付き会社概要テーブル + 支援対象タグ
- **CONTACT** — フォーム型UI（名前/メール/会社名/内容）、プライバシーポリシーリンク

### プライバシーポリシーページ (`privacy.html`)
- 全8条の個人情報保護方針
- お問い合わせ窓口情報

### 共通機能
- **固定ヘッダー** — ミントホワイト背景 + バックドロップブラー
- **ウォーターマーク** — 各セクションに大型アウトライン英字ラベル（SERVICE / PHILOSOPHY / ABOUT US / CONTACT）
- **スクロールリビール** — IntersectionObserverによるフェードインアニメーション
- **モバイルナビゲーション** — ハンバーガーメニュー（800px以下）
- **装飾キューブ** — ヒーローセクションの幾何学的装飾要素
- **レスポンシブデザイン** — モバイル〜デスクトップ完全対応
- **prefers-reduced-motion対応**

## デザインシステム

### カラーパレット
- **Deep Green**: `#13382A` ~ `#38A87D`（キーカラー）
- **Cool Mint**: `#F4F8F6`（メイン背景）
- **White**: `#FFFFFF`（フォーム・カード背景）
- **Text**: `#1E2D28`（メインテキスト）、`#5A6B63`（補助テキスト）

### フォント
- Inter + Noto Sans JP + Noto Serif JP（Google Fonts CDN）

### 特徴的なデザイン要素
- **ウォーターマークラベル**: 各セクション背景に大型アウトラインテキスト（8〜15%透過）
- **スタッガードレイアウト**: サービスセクションの左右交互配置
- **グリーン背景セクション**: PHILOSOPHYで使用
- **装飾キューブ**: 45度回転した小さな正方形が浮遊
